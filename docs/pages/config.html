Configuration

Typeahead-standalone.js provides several configuration options to fine-tune and customize autocompletion. We list all available config options below. Of these, only the first two are required whereas the rest use sensible default values.

1. input (Required)

This is the DOM input element typeahead will attach itself to. It is a required property and an error is thrown if it is missing

Javascript HTML
JS 
typeahead({
  input: document.querySelector('.myInput'),  // required
})
HTML 
<input id="typeahead" placeholder="myInput">

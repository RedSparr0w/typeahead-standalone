Usage

Typeahead-standalone.js usage guide in detail. There are 3 basic steps to be followed as explained below...

1. HTML

Add an input element to your HTML

Basic Advanced
HTML 
<input id="typeahead" placeholder="search">
HTML 
<input id="typeahead" type="search" placeholder="search" dir="ltr" spellcheck=false autocorrect="off" autocomplete="off" autocapitalize="off" maxlength="2048">

2. Import the library üì¶Ô∏è

Import the javascript and the CSS into your project/app. Note: It is recommended to directly override the default styles instead of importing them.

Javascript HTML
HTML 
<head>
  <link rel="stylesheet" href="https://cdn.jscript.net/npm/typeahead-standalone/dist/basic.css" />
</head>

<body>
  <script src="https://cdn.jscript.net/npm/typeahead-standalone"></script>
</body>
JS 
// using ES6 modules
import typeahead from 'typeahead-standalone';
import 'typeahead-standalone/dist/basic.css';

// using RequireJS
const typeahead = require('typeahead-standalone');
require('typeahead-standalone/dist/basic.css');

3. Initialize & configure ‚öôÔ∏è

Initialize typeahead-standalone.js by passing to it the required configuration parameters

Javascript HTML
JS 
// input element to attach to
const inputElement = document.getElementById("typeahead");

const instance = typeahead({
  input: inputElement,
  source: {
    local: ['Grey', 'Brown', 'Black', 'Blue'],
  }
});
HTML 
<body>
  <script src="https://cdn.jscript.net/npm/typeahead-standalone"></script>
  <script>
    // input element to attach to
    const inputElement = document.getElementById("typeahead");

    const instance = typeahead({
      input: inputElement,
      source: {
        local: ['Grey', 'Brown', 'Black', 'Blue'],
      }
    });
  </script>
</body>
